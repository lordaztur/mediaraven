# Começando

Pra ter o MediaRaven rodando você precisa de **três coisas**:

1. **Instalar** o código + dependências Python — [Instalação](install.md)
2. **Subir o servidor Bot API local** + criar o bot no @BotFather — [Configuração](config.md)
3. **Preencher o `.env`** com token, IDs e paths — [Configuração](config.md)

Depois é só `python mediaraven.py` e mandar um link no Telegram.

## Pré-requisitos resumidos

- Python 3.11+
- ffmpeg (pra mixagem de áudio/vídeo)
- Docker (pro servidor Bot API local)
- Opcional: Deno (pra bypass JS do YouTube), Firefox (pra reaproveitar cookies)
